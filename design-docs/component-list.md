# HDD UI 组件库 - 基础组件清单

> 参考 Element Plus、Ant Design Vue 等主流UI框架，梳理基础组件清单

## 一、通用组件 (General)

### 1. Button 按钮
- **核心功能**：用于触发一个操作，如提交表单、确认删除等
- **支持类型**：primary（主色）、success（成功）、warning（警告）、danger（危险）、info（信息）
- **支持变体**：plain（幽灵按钮）、text（文字按钮）、dashed（虚线按钮）
- **支持尺寸**：sm（小）、md（中）、lg（大），默认sm
- **支持状态**：loading（加载中）、disabled（禁用）
- **附加功能**：icon（图标按钮）、block（块级按钮）、round（圆角按钮）

### 2. Icon 图标
- **核心功能**：提供一套常用图标，支持自定义图标
- **支持功能**：修改颜色、尺寸、旋转、翻转、旋转动画（spin）
- **使用方式**：内置图标、自定义SVG图标、图标库集成

### 3. Link 链接
- **核心功能**：文字链接，支持不同状态和类型
- **支持类型**：默认、主色、成功、警告、危险
- **支持状态**：禁用、下划线控制

### 4. Text 文字
- **核心功能**：文本展示，支持不同样式和大小
- **支持类型**：标题、正文、辅助文字
- **支持样式**：颜色、加粗、斜体、删除线

### 5. Space 间距
- **核心功能**：控制组件之间的间距
- **支持方向**：横向、纵向
- **支持尺寸**：小、默认、大、超大、自定义

---

## 二、布局组件 (Layout)

### 1. Layout 布局
- **核心功能**：页面整体布局，提供 Header、Sider、Content、Footer
- **组件组成**：Layout（容器）、Header（页头）、Sider（侧边栏）、Content（内容区）、Footer（页脚）
- **支持布局**：上中下三栏、左右两栏、嵌套布局
- **API 属性**：
  - HddLayout：`has-sider`（是否包含侧边栏，默认 false）
  - HddHeader：`height`（高度，默认 60px）
  - HddFooter：`height`（高度，默认 60px）
  - HddSider：`width`（宽度，默认 200px）、`collapsed`（是否折叠，默认 false）

### 2. Container 容器
- **核心功能**：页面容器组件，用于包裹页面内容
- **支持功能**：最大宽度限制、居中对齐、内边距

### 3. Grid 栅格
- **核心功能**：基于24列栅格系统，进行响应式布局
- **组件组成**：Row（行）、Col（列）
- **支持功能**：列偏移、栅格排序、响应式断点

### 4. Flex 弹性布局
- **核心功能**：基于Flexbox的布局组件
- **组件组成**：Flex（容器）、FlexItem（子项）
- **支持功能**：方向、对齐方式、换行、间距、比例分配

### 5. Divider 分割线
- **核心功能**：区隔内容的分割线
- **支持类型**：水平分割线、垂直分割线
- **支持样式**：实线、虚线、带文字分割线
- **API 属性**：
  - `direction`（方向，默认 horizontal，可选 vertical）
  - `content-position`（文字位置，默认 center，可选 left/center/right）
  - `dashed`（是否虚线，默认 false）
  - `border-color`（边框颜色，默认继承主题色）
  - `text-color`（文字颜色，默认继承主题色）
  - `margin`（外边距，默认 16px 0）

## 三、导航组件 (Navigation)

### 1. Menu 菜单
- **核心功能**：提供多级菜单导航
- **支持模式**：水平菜单、垂直菜单、折叠菜单
- **支持功能**：当前选中、展开/收起、主题（浅色/深色）

### 2. Tabs 标签页
- **核心功能**：在不同内容区域间切换
- **支持类型**：默认、卡片、简约
- **支持功能**：可关闭标签、添加标签、位置（上/下/左/右）

### 3. Breadcrumb 面包屑
- **核心功能**：显示当前页面在系统层级中的位置
- **支持功能**：分隔符自定义、最后一项可高亮

### 4. Pagination 分页
- **核心功能**：数据分页展示
- **支持功能**：页码、每页条数选择、快速跳转、总条数显示

### 5. Steps 步骤条
- **核心功能**：引导用户按步骤完成任务
- **支持状态**：进行中、已完成、等待、错误
- **支持方向**：水平、垂直

### 6. Affix 固钉
- **核心功能**：将元素固定在页面特定位置
- **支持功能**：距离顶部/底部、相对容器定位

### 7. Anchor 锚点
- **核心功能**：用于跳转到页面指定位置
- **支持功能**：平滑滚动、高亮当前锚点

---

## 四、数据录入组件 (Data Entry)

### 1. Input 输入框
- **核心功能**：单行文本输入
- **支持类型**：默认、密码、搜索、文本域
- **支持功能**：清空按钮、前缀/后缀、字数统计、尺寸、状态

### 2. InputNumber 数字输入框
- **核心功能**：数字输入，支持加减按钮
- **支持功能**：最小值/最大值、步长、精度、格式化

### 3. Select 选择器
- **核心功能**：从预设选项中选择一个或多个值
- **支持类型**：单选、多选、分组、可搜索、可清空、可创建
- **支持功能**：远程搜索、选项加载中、自定义选项模板

### 4. Cascader 级联选择器
- **核心功能**：从一组相关联的数据集合进行选择
- **支持功能**：多选、可搜索、禁用选项、自定义渲染

### 5. Radio 单选框
- **核心功能**：从多个选项中选择一个
- **组件组成**：Radio（单选框）、RadioGroup（单选框组）
- **支持功能**：按钮样式、禁用、尺寸

### 6. Checkbox 多选框
- **核心功能**：从多个选项中选择多个
- **组件组成**：Checkbox（多选框）、CheckboxGroup（多选框组）
- **支持功能**：按钮样式、禁用、全选/半选、尺寸

### 7. Switch 开关
- **核心功能**：开关状态切换
- **支持功能**：文字描述、禁用、尺寸、加载中

### 8. Slider 滑块
- **核心功能**：在范围内选择数值
- **支持类型**：单滑块、双滑块
- **支持功能**：步长、刻度、输入框、禁用

### 9. DatePicker 日期选择器
- **核心功能**：选择日期或日期范围
- **支持类型**：日期、日期时间、日期范围、月份、年份
- **支持功能**：日期格式化、禁用日期、快捷选项

### 10. TimePicker 时间选择器
- **核心功能**：选择时间
- **支持类型**：时间、时间范围
- **支持功能**：步长、禁用时间、格式

### 11. DateTimePicker 日期时间选择器
- **核心功能**：选择日期和时间
- **支持类型**：日期时间、日期时间范围

### 12. Upload 上传
- **核心功能**：文件上传
- **支持类型**：点击上传、拖拽上传、头像裁剪上传
- **支持功能**：文件数量限制、类型限制、大小限制、进度条

### 13. Rate 评分
- **核心功能**：评分组件
- **支持功能**：星星数量、半星、允许取消、只读、自定义图标

### 14. ColorPicker 颜色选择器
- **核心功能**：颜色选择
- **支持功能**：预设颜色、透明度、颜色格式

### 15. Transfer 穿梭框
- **核心功能**：在两个选项框之间穿梭选择
- **支持功能**：搜索、自定义渲染、全选

### 16. Form 表单
- **核心功能**：表单数据收集、验证、提交
- **组件组成**：Form（表单容器）、FormItem（表单项）
- **支持功能**：表单验证、标签对齐、行内表单、标签宽度

---

## 五、数据展示组件 (Data Display)

### 1. Table 表格
- **核心功能**：展示结构化数据
- **支持功能**：排序、筛选、分页、选择、展开行、树形数据、可编辑、固定列、斑马纹、边框

### 2. Tree 树形控件
- **核心功能**：展示层级结构数据
- **支持功能**：多选、可拖拽、懒加载、自定义节点、复选框

### 3. TreeSelect 树形选择器
- **核心功能**：树形结构的选择器
- **支持功能**：多选、可搜索、可清空、自定义节点

### 4. Tag 标签
- **核心功能**：标签展示和筛选
- **支持类型**：默认、圆型、边框
- **支持功能**：可关闭、自定义颜色、尺寸

### 5. Badge 徽标
- **核心功能**：在图标或按钮上添加徽章
- **支持类型**：数字徽章、圆点徽章、自定义内容
- **支持功能**：最大数、位置

### 6. Avatar 头像
- **核心功能**：用户头像展示
- **支持类型**：图标头像、文字头像、图片头像
- **支持功能**：尺寸、形状、分组

### 7. Carousel 走马灯
- **核心功能**：图片轮播
- **支持功能**：自动播放、箭头、指示器、切换动画

### 8. Collapse 折叠面板
- **核心功能**：将内容折叠/展开
- **支持功能**：手风琴模式、自定义标题

### 9. Timeline 时间线
- **核心功能**：时间轴展示
- **支持类型**：时间、状态、自定义图标
- **支持方向**：垂直、水平

### 10. Tooltip 文字提示
- **核心功能**：鼠标悬停时显示提示
- **支持功能**：位置、延迟、禁用、触发方式

### 11. Popover 弹出框
- **核心功能**：点击/悬停时弹出内容
- **支持功能**：位置、触发方式、嵌套

### 12. Popconfirm 气泡确认框
- **核心功能**：点击元素前确认
- **支持功能**：位置、自定义按钮

### 13. Card 卡片
- **核心功能**：通用卡片容器
- **支持功能**：标题、操作栏、阴影、边框

### 14. Descriptions 描述列表
- **核心功能**：键值对形式展示详情
- **支持功能**：列数、边框、布局

### 15. Empty 空状态
- **核心功能**：空数据状态展示
- **支持功能**：自定义图片、描述、操作按钮

### 16. Result 结果页
- **核心功能**：操作结果反馈页面
- **支持类型**：成功、失败、警告、信息、404、403、500
- **支持功能**：自定义图标、标题、描述、操作按钮

### 17. Skeleton 骨架屏
- **核心功能**：加载时的占位图
- **支持类型**：文字、头像、图片、列表、自定义组合
- **支持功能**：动画、行数

### 18. Image 图片
- **核心功能**：图片展示和预览
- **支持功能**：懒加载、占位图、加载失败、预览、缩放

## 六、反馈组件 (Feedback)

### 1. Alert 警告提示
- **核心功能**：展示警告或提示信息
- **支持类型**：成功、警告、错误、信息
- **支持功能**：可关闭、图标、描述文字

### 2. Message 消息提示
- **核心功能**：轻量级的全局提示
- **支持类型**：成功、警告、错误、信息、加载中
- **支持功能**：显示时长、位置、关闭按钮

### 3. MessageBox 弹框
- **核心功能**：确认框、提示框、输入框
- **支持类型**：alert、confirm、prompt
- **支持功能**：自定义按钮、回调函数

### 4. Notification 通知
- **核心功能**：右上角通知提醒
- **支持类型**：成功、警告、错误、信息
- **支持功能**：位置、时长、关闭按钮、自定义内容

### 5. Drawer 抽屉
- **核心功能**：侧边滑出面板
- **支持功能**：方向（左/右/上/下）、尺寸、遮罩、关闭按钮

### 6. Modal / Dialog 对话框
- **核心功能**：模态对话框
- **支持功能**：标题、内容、底部按钮、全屏、遮罩、可拖拽

### 7. Progress 进度条
- **核心功能**：显示操作进度
- **支持类型**：线性进度条、圆形进度条
- **支持功能**：百分比、状态（进行中/成功/失败/警告）、条纹、动画

### 8. Loading 加载
- **核心功能**：加载状态指示器
- **支持类型**：全屏加载、局部加载、自定义加载图标
- **支持功能**：文字提示、遮罩

---

- **核心功能**：回到页面顶部

## 开发优先级建议

### 第一阶段（基础）
- **通用**：✅ Button、✅ Icon
- **布局**：✅ Layout、✅ Grid、✅ Flex、✅ Divider
- **数据录入**：Input、InputNumber、Select、Radio、Checkbox、Switch
- **数据展示**：Tag、Badge、Avatar、Tooltip
- **反馈**：Alert、Message、Progress、Loading

### 第二阶段（核心）
- **通用**：Link、Text、Space
- **导航**：Tabs、Breadcrumb、Pagination、Steps
- **数据录入**：DatePicker、TimePicker、Upload、Form
- **数据展示**：Table、Tree、Card、Collapse
- **反馈**：Modal、MessageBox、Notification

### 第三阶段（完善）
- **导航**：Menu、Affix、Anchor
- **数据录入**：Cascader、Slider、Rate、ColorPicker、Transfer
- **数据展示**：Descriptions、Empty、Result、Skeleton、Image
- **反馈**：Drawer、Popover、Popconfirm
- **其他**：Backtop、Scrollbar
